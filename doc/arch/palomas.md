## Cómo crear una paloma

Usar el comando global `/paloma` o copiar este template. Las palomas son "tareas suspendidas" que esperan a que se den las condiciones para ejecutarse (ej: "cuando el bot tenga 30 días de paper trading, revisar hit rate").

## Archivos relacionados

- `INDEX.md` — punto de entrada del proyecto
- `AGENTS.md` — variable `$PALOMA_MAIN_PLAN` apunta acá
- `DILIGENCIA.md` — metodología
