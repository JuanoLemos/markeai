# /rm — Revisar Roadmap

Revisa `doc/documentos/roadmap.md`.

## Argumentos
`/rm` — revisa roadmap completo

## Qué hace
1. Lee `doc/documentos/roadmap.md`
2. Lista todos los items PENDIENTE con su prioridad y fase
3. Lista items DONE recientes
4. Identifica bloqueos (dependencias entre fases)
5. Reporta resumen ejecutivo
