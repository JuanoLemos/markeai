# /focoux — Enfocar mente en la experiencia de usuario

Cambia la inclinación del agente hacia pensamiento **de experiencia** (trading UX, alertas, dashboard, flujo de operación, estrategias, notificaciones, monitoreo).

No solo lee guías de UX — el agente debe:
1. Revisar el proyecto con ojos de trader: ¿el dashboard muestra la info correcta? ¿las alertas son útiles? ¿el flujo de operación es claro?
2. Identificar mejoras de experiencia: métricas visibles, alertas configurables, reportes, visualización de estrategias
3. Leer `guias/guia_uso.md` para entender el flujo actual
4. Considerar: resúmenes diarios, notificaciones push, reportes semanales, modo replay, visualización de estrategias en dashboard
5. Reportar: próximos pasos de UX + mejoras detectadas + items pendientes
